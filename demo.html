Jump

Make the active target jump.
Say "Hi"

The active target will activate the voice synthesizer module to say hello.
Compliment

The active target will say a compliment without being scared.
Hug

This forces the target to make a hug gesture without feeling any shyness.

 
# preparation phase
#                          x      y      z  [time]
await body.ankle.rotate   (0,     0, 30deg, 0.3sec)
await body.knees.translate(0, -10cm, 0,     0.3sec)
​
# short pause
await time.sleep(1sec)
​
# launch phase
await body.knees.translate(0,  20cm, 0)
​
$ > ./jump
# say(text[, duration, pitch])
await body.voice.say("Hi!")
​
$ > ./hi
# Prepare the waving gesture
await body.right_hinge.rotate(50deg, 0, 0)
await body.right_shoulder.rotate(0, 0, 40deg)
​
# say(text[, duration, pitch])
await body.voice.say("You are looking lovely today!")
​
# Wave
for _ in range(5):
    await body.right_hinge.rotate(10deg, 0, 0, 0.3sec)
    await body.right_hinge.rotate(10deg, 0, 0, 0.3sec)
​
$ > ./compliment
# Prepare the back of the body for a hug
await body.sternum.rotate(5deg, 0, 0, 0.3sec)
await body.spine.rotate(20deg, 0, 0, 0.3sec)
​
# Rotate the arms from the shoulders
await body.hinges.rotate(0, 40deg, 0, 0.5sec)
await body.shoulders.rotate(0, 15deg, 40deg, 0.5sec)
​
# Slowly get the arms wrapped around one's body
await body.shoulders.rotate(0, 45deg, 40deg, 3sec)
​
# Wait and enjoy?
await time.sleep(3sec)
​
# Slowly stop having the arms wrapped around one's body
await body.shoulders.rotate(0, 15deg, 40deg, 3sec)
​
# Reset the body to the IDLE position
await body.reset(0.5sec)
​
$ > ./hug
